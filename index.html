Project 1 - Zelda

--- ReadMe ---

Warning: Some tiles flash colors. Mild epilepsy warning

Controls:

  • "WASD" or Arrow Keys to move
  • Run by holding "Left Shift"
  • A Attack with "Spacebar"
  • B Attack with "Left Ctrl"
  • switch A weapon with numbers "1-2"
  • switch B weapon with "Left Shift" and numbers "1-2"
  • Load Main level with "3"
  • Load Custom level with "4"
  • Enable Godmode by pressing "0"
